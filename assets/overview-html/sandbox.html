Trying the APIs

Each API can be tried using the API Explorer. You can use the sample data mentioned above and to try and test the APIs. To try the APIs, click on the particular API in the menu. You will see a tab that says "Try It Out". This section provides the ability to make API calls.

Authentication and consent

To enable live interaction with APIs in the sandbox, we've provided an implementation of the authentication and consent app. APIs such as customer or order require explicit end-user consent to access his/her profile information. Follow these steps to authorize before you can fire the APIs.
